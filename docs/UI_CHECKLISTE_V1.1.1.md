# ✅ UI‑Checkliste für Rekonstruktion
Status: Draft
Version: 1.1.1
Datum: 2025-11-11
Autor: Engineering

## Zusammenfassung
Pragmatische Liste aller zu prüfenden UI‑Elemente vor Umsetzung.

## Details
- Header
  - Hintergrund `#FFFFFF`, kein `backdrop-blur`, korrekte Höhe/Sticky‑Verhalten
  - Navigation: Items, Reihenfolge, aktive States
- Footer
  - Hintergrund `#FFFFFF`, keine dunkle Abschlusslinie, Links/Legal korrekt
- Chat
  - Icon austauschen (Lucide monochrom), FAB Position, Panel Öffnen/Schließen
  - DSGVO Consent, ARIA, Fokus, ESC
- Seiten
  - Container/Gutter konsistent; Section‑Spacing gemäß Styleguide

- Abnahme durch QA und Accessibility‑Review
