## üí° DESIGN-BEST-PRACTICES (NEU)

### DO's ‚úÖ

#### Visual Design

- **Multi-Layer Shadows:** Mindestens 2-3 Schatten-Ebenen
- **Glassmorphism:** `backdrop-blur-xl` + `bg-background/80`
- **Gradients:** Subtil, nicht aufdringlich
- **Hover-States:** Smooth mit 300-500ms duration

- **Spacing:** 4px-Grid-System √ºberall
- **Rounded Corners:** Mindestens `rounded-xl` (1rem)

- **Performance:** Lazy-load alles √ºber 100KB
- **Responsive:** 5 Breakpoints (xs, sm, md, lg, xl)

### DON'Ts ‚ùå

#### Visual Don'ts

- **Flat Design:** Keine Depth = Unprofessionell
- **Single Shadows:** Mindestens multi-layer
- **Direct Colors:** `text-white` = VERBOTEN
- **Harte Transitions:** < 200ms = zu schnell
- **Keine Hover-States:** = Dead Interface
- **Comic Sans:** (Offensichtlich, aber...)

#### Technical Don'ts

- **Inline Styles:** `style={{ ... }}` = VERBOTEN
- **Magic Numbers:** Spacing ohne System
- **!important:** Nur in absoluten Notf√§llen
- **Z-Index Chaos:** System verwenden (10, 20, 30, ...)
